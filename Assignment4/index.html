Assignment Description:

Design inspiration. Each student will choose 1 movement, artist, or designer for a 3-5 min presentation to the class. Share a brief introduction to the movements, artists, or designers and at least 3 images for each.

You MUST create your slidedeck using HTML, CSS + host on your class site. Each slide could be a page you link to from the slide before. Use an external styling sheet to keep the styling of your presentation uniform. Your slide deck website must include:
  • It's own directory + file path linked from your class site home page.
  • Multiple HTML pages controlled by 1 external styling sheet.
  • Some navigation with links to relative file path to go to the next "slide".
  • At least one instance of CSS classes + one id.
  • An "images" or "assets" directory to hold your media.
  • A "pages" or "slides" directory to hold your pages.

Post a link to your Design Thinking Presentation Web Site (Slide Deck) by midnight on Tuesday.
